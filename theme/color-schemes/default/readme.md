```
FF ULTIMA
Default Edition
FF Ultima's Default Color scheme - Full Moon (Dark Mode) and Dusky (Light Mode)
```

To use this color scheme:
- Navigate to `about:config` page.
- Search for `user.theme`.
- Turn on `user.theme.0.default`
- Enjoy.
