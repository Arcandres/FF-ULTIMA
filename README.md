### Features
‚ñ∫ NO extensions used, NO sidebar needed<br>
‚ñ∫ `Vertical Tabs` and a `One Line Nav Bar`<br>
‚ñ∫ a `Mini Button Bar` can pin/unpin 15 buttons, just drag and drop, simply remove all buttons and it is gone<br>
